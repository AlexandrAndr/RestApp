Руководство пользователя

Приложение "Полицейский" (далее по тексту приложение) преднозначено для
генерации рандомных и последовательных регистрационных знаков легковых транспортных средств РФ.

Принцип работы приложения

Приложение генерирует рандомный автомобильный номер и выдает последующие
за ним номера. Все автомобильные номера сохраняются в БД.

Примечание для проверяющих приложение: приложение имеет два профиля "dev" - режим разработки
и "prod" - режим работы программы пользователем. В режиме "dev" слова сохраняются в БД "PostgreSQL",
в режиме "prod" - слова сохраняются в БД "Н2". Сделано это для того, чтобы пользователю не приходилось
создавать и подключать БД. По умолчанию установлен профиль - "prod".

Запуск и работа с приложением

Для тестирования приложения откройте фаил "Полицейский" с расширением ".bat" и
приложение автоматически запустится.
После загрузки приложения откройте браузер и укажите следующий URL:

http://localhost:8080/number/random

Приложение сгенерирует Вам автомобольный номер.
После генерации номера можно получить следующий за ним номер. Для этого
укажите  URL следующем формате:

http://localhost:8080/number/next?number=Х001ХХ 116 RUS,

где Х- это русские символы серии сгенерированного номера, а 001 - это номер.
